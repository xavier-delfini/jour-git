Les différentes commandes Git

Commandes Explication
config --global user.name "[nom]" Définit le nom que vous voulez associer à toutes vos opérations de commit
config --global user.email "[adresse email]" Définit l'email que vous voulez associer à toutes vos opérations de commit
config --global color.ui auto Active la colorisation de la sortie en ligne de commande
init [nom-du-projet] Crée un dépôt local à partir du nom spécifié
clone [url] Télécharge un projet et tout son historique de versions
status Liste tous les nouveaux fichiers et les fichiers modifiés à commiter
diff Montre les modifications de fichier qui ne sont pas encore indexées
add [fichier] Ajoute un instantané du fichier, en préparation pour le suivi de version
diff --staged Montre les différences de fichier entre la version indexée et la dernière version
reset [fichier] Enleve le fichier de l'index, mais conserve son contenu
commit -m "[message descriptif]" Enregistre des instantanés de fichiers de façon permanente dans l'historique des versions
branch Liste toutes les branches locales dans le dépôt courant
branch [nom-de-branche] Crée une nouvelle branche
checkout [nom-de-branche] Bascule sur la branche spécifiée et met à jour le répertoire de travail
merge [nom-de-branche] Combine dans la branche courante l'historique de la branche spécifiée
branch -d [nom-de-branche] Supprime la branche spécifiée
rm [fichier] Supprime le fichier du répertoire de travail et met à jour l'index
rm --cached [fichier] Supprime le fichier du système de suivi de version mais le préserve localement
mv [fichier-nom] [fichier-nouveau-nom] Renomme le fichier et prépare le changement pour un commit
ls-files --other --ignored --exclude-standard Liste tous les fichiers exclus du suivi de version dans ce projet
stash Enregistre de manière temporaire tous les fichiers sous suivi de version qui ont été modifiés ("remiser son travail")
stash pop Applique une remise et la supprime immédiatement
stash list Liste toutes les remises
stash drop Supprime la remise la plus récente
log Montre l'historique des versions pour la branche courante
log --follow [fichier] Montre l'historique des versions, y compris les actions de renommage, pour le fichier spécifié
diff [premiere-branche]...[deuxieme-branche] Montre les différences de contenu entre deux branches
show [commit] Montre les modifications de métadonnées et de contenu inclues dans le commit spécifié
reset [commit] Annule tous les commits après `[commit]`, en conservant les modifications localement
reset --hard [commit] Supprime tout l'historique et les modifications effectuées après le commit spécifié
fetch [nom-de-depot] Récupère tout l'historique du dépôt nommé
merge [nom-de-depot]/[branche] Fusionne la branche du dépôt dans la branche locale courante
push [alias] [branche] Envoie tous les commits de la branche locale vers GitHub
pull Récupère tout l'historique du dépôt nommé et incorpore les modifications
